# record-student-seat
- Vue3、pinia(狀態儲存工具)
- 此工具為紀錄學生的位置與學習記錄，提供老師使用拖拉方式(因應平板需求)安排學生座位與記錄學生學習情形

## Project setup

```
yarn install
```
